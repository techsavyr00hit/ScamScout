# ScamScout 🚨

ScamScout is a browser extension that helps users identify fake or suspicious job postings in real time.

## ✨ Features

* Detects scam job listings
* Highlights suspicious content
* Provides a simple risk indicator

## 🚀 How to Use

1. Install the extension
2. Open a job posting
3. ScamScout automatically analyzes the content
4. View the scam risk alert

## 🛠️ Tech (example)

* JavaScript
* Browser Extension APIs
* Basic AI / rule-based detection

## ⚠️ Disclaimer

ScamScout only provides risk indications and may not be 100% accurate. Always verify job postings independently.
